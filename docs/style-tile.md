# ETI Style Tile

## Direction
- Mood: Scholarly + cinematic + modern warmth
- Reference energy: Minerva-level clarity, but less corporate and more human

## Typography
- Display serif: `Fraunces`
- UI/body sans: `Plus Jakarta Sans`
- Rules:
  - Headline tracking slightly tight
  - Body line height generous
  - Eyebrows uppercase with wider letter spacing

## Color System
- Paper: `#F4EFE6`
- Surface: `#FFFDF8`
- Ink: `#161719`
- Muted text: `#4E5055`
- Deep academic tone: `#0F3B42`
- Accent (CTA): `#BA5A2D`
- Accent hover: `#9D4219`
- Divider line: `#D9D0C1`

## Components
- Primary button: rounded pill, warm accent fill, subtle lift on hover
- Secondary button: outlined neutral, darkens on hover
- Cards: soft border, warm surface, mild shadow
- Quotes: no heavy ornamentation, clean confidence

## Imagery
- Hero: full-bleed cinematic (or muted loop video)
- Supporting blocks: documentary-style learning moments
- Treatment:
  - Favor warm natural light
  - Keep faces and interactions prominent
  - Avoid stock-photo stiffness

## Motion
- Page-load reveal with staggered delays
- Minimal transform distance (`translateY` small)
- No looping novelty animations

## Spatial Rhythm
- Generous top/bottom padding between major sections
- Tight heading-to-body spacing for clarity
- Consistent max-width rhythm across all sections
